# fastApiLab
Laboratorio FastAPI

Paso a paso
1. Creación de carpeta con nombre deseado (laboratorioFastApi)
2. Creacion de entorno virtual con: python -m venv virtualFast
3. Activacion de entorno virtual con: virtualFast\Scripts\Activate
4. Realizar la instalacion del proyecto con: pip install fastapi
5. Se necesita un servidor ASGI para la produccion con: pip install ”uvicorn[standard]”
6. Se crea el archivo main.py
7. Dentro de main.py se encuentra el codigo respectivo con todas sus importaciones, validaciones de datos y funciones necesarias para la obtencion y publicacion de datos. Ademas de la explicacion del codigo
8. Para correr el servidor es necesario el comando: uvicorn main:app --reload
